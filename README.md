### 🚀 Cómo utilizar
1. Clona el repositorio:
```bash
git clone https://github.com/Jevin1127/Ejercicios-en-C-.git
```

2. Explora por categorías:

    **Basicos - Para fundamentos de C#**

    **Avanzados - Ejercicios complejos**

    **ProyectoFinal - Sistema completo**

3. Recomendado
```bash
cd Proyecto\ final
dotnet run
```
